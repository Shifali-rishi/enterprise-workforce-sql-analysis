# Enterprise Workforce & Project Analytics

## 📌 Project Overview
This project demonstrates SQL-based analysis on an enterprise workforce dataset, including employees, departments, projects, and work allocation.

## 🗂️ Tables Used
- employees
- department
- project
- works_on

## 🎯 Objectives
- Analyze employee distribution
- Track project allocation
- Evaluate department performance
- Identify unassigned resources

## 🛠️ Skills Demonstrated
- SQL Joins
- Subqueries
- Aggregations
- Group By & Having
- Real-world business queries

## 📊 Key Queries
- Employees by department
- Salary analysis
- Project distribution
- Unassigned employees detection

## 🚀 Outcome
This project showcases strong SQL problem-solving skills and real-world data analysis capability.
